# 🎯 Jogo do Número Secreto

> Projeto simples desenvolvido para praticar lógica de programação com JavaScript.

---

## 📚 Sobre o projeto

Este projeto foi desenvolvido durante o curso de lógica de programação com JavaScript da Alura.
O objetivo é reforçar conceitos básicos de programação por meio da criação de um jogo interativo de adivinhação.
O jogo consiste em descobrir um número secreto gerado pelo sistema, recebendo dicas se o valor é maior ou menor a cada tentativa. :contentReference[oaicite:0]{index=0}

---

## 🧠 Objetivo de aprendizado

Durante o desenvolvimento deste projeto, foram praticados:

- Variáveis e tipos de dados
- Estruturas condicionais (if/else)
- Interação com o usuário (alert, prompt)
- Lógica de comparação
- Fluxo de execução

---

## ⚙️ Como funciona

1. O sistema gera um número secreto
2. O usuário informa um palpite
3. O jogo responde se o número é maior ou menor
4. O processo se repete até o acerto
5. Ao acertar, o jogo exibe uma mensagem de sucesso
